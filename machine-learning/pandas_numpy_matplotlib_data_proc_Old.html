Data Wrangling using Pandas and Numpy

Introduction

According to World Health Organization (WHO), cardiovascular diseases (CVDs) are the number 1 cause of death globally, taking an estimated 17.9 million lives each year and affecting the quality of life of a large number of people worldwide. Prerequisites of the treatment of these types of disease involve proper diagnosis method to identify its occurrence and its type. Diagnosis of such diseases involve in-depth analysis of a large number of parameters. In this demo Pandas and Numpy is used for analysing one such dataset called A1_heart_disease_dataset.

Pandas

Pandas is a widely used, robust and flexible open source tool for data analysis and manipulation. Pandas is used for exploratory data analysis, and drawing conclusions from data. The fundamental of data analysis contains 4 steps, Data Collection, Data Wrangling, Exploratory Data Analysis (EDA), and Drawing Conclusions.

Pandas dundamental datastructures are DataFrame, Series and Index. It is also possible to use numpy arrays with pandas for short CSVs.

Pandas DataFrame is the basic building block of Data analysis, The DataFrame class builds upon the Series class and can have many columns, each with its own data type; we can think of it as representing the spreadsheet as a whole.

In the following example Python requests is used to U.S. Geological Survey's API to grab a JSON object of earthquake data and convert it to a pandas.DataFrame

USGS API: https://earthquake.usgs.gov/fdsnws/event/1/

The json response can be used to construct a dataframe. Similarly Pandas dataframe can be constructed from various sources such as excel, tab or comma seperated files etc.

Numpy

NumPy builds on (and is a successor to) the successful Numeric array object. Its goal is to create the corner-stone for a useful environment for scientific computing. Numpy provides two fundamental objects, an N-dimensional array object (ndarray) and a universal function object (ufunc).
