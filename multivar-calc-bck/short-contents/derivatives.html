Partial Derivative

We taking partial derivative of a function with respect to one variable, treat the other independent variables as constants. $ \frac{\partial f}{\partial x} $ denotes a partial derivative of the function f with respect to variable x.

Example Given $ f = 3x \ \sin \ t, \ \text{find} \frac{\partial f}{\partial x} \ \text{find} \frac{\partial f}{\partial t} $

When finding $\frac{\partial f}{\partial x}$, the other independent variable has to be treated as a constant $$ \frac{\partial f}{\partial x} = \frac{\partial }{\partial x}3x \sin \ t = 3 \sin \ t \frac{\partial }{\partial x}x = 3 \ \sin t(1) = 3 \ Sin \ t $$

Similarly, when finding $\frac{\partial f}{\partial t}$, we treat the independent variable x as constant $$ \frac{\partial f}{\partial t} = \frac{\partial }{\partial t}3x \sin t = 3x \frac{\partial }{\partial t} \sin t = 3x \ (cos t) = 3x \ Cos \ t $$

Now we want to find the second order partial derivative, we need to take one partial derivative at a time.

Chain Rule with Multiple Variable

The Chain Rule applied to Single Variable works as follows $$ \frac{\partial f}{\partial t} = \frac{\partial f}{\partial u} \frac{\partial u}{\partial t} $$ Chain rule applied to single variable $ \frac{\partial }{\partial t} \sin(3t^{2})$ Let u = $3t^{2}$ and $f = \sin(3t^{2})$ In this example the chain rule gives the following $$ \frac{\partial}{\partial t} \sin(3t^{2}) = \frac{\partial f}{\partial t} $$ $$ (\frac{d}{du} \sin \ u )( \frac{d}{dt} 3t^2 ) $$ $$ (\cos u)(6t) = 6t cos(3t^2) $$

The Chain Rule applied to Multiple Variable works as follows. If f is a function of two independent variables f = f(x,y) and if these independent veriables, x and y themselves functions of two more independent variables x = x(s, t) and y = y(s, t), then we can find the partial derivative of f W.R.T s and t can be found using the Chain Rule $$ \frac{\partial f}{\partial s} = \frac{\partial f}{\partial x} \frac{\partial x}{\partial s} + \frac{\partial f}{\partial y} \frac{\partial y}{\partial s}$$ $$ \frac{\partial f}{\partial t} = \frac{\partial f}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial f}{\partial y} \frac{\partial y}{\partial t}$$

Example: Given $$z = x^3y^4, x = s \cos t, \text{and} \ y = s \sin t , \ text{find} \ \ \frac{\partial z}{\partial s} \ \text{and} \ \frac{\partial z}{\partial t} $$ Applying the Chain Rule: $$ \frac{\partial z}{\partial s} = \frac{\partial z}{\partial x} \frac{\partial x}{\partial s} + \frac{\partial z}{\partial y} \frac{\partial y}{\partial s} $$ $$ = \Big( \frac{\partial}{\partial x} \ x^{3}y^{4} \Big) \Big( \frac{\partial}{\partial s} s \cos t \Big) + \Big( \frac{\partial}{\partial y} x^{3}y^{4} \Big) \Big( \frac{\partial}{\partial s} s \sin t \Big) $$

$$ \frac{\partial z}{\partial s} = (3x^{2}y{4})(\cos t) + (4x^3y^3) (\sin t) = (3x^{2}y^{4}) \cos t + (4x^3y^3) \sin t $$ Now plugging in $x = s \ \text{and} \ y = s \sin t$ $$ \frac{\partial z}{\partial s} = 3(s \cos t)^{2} (s \sin t)^{4} \cos t + 4(s \cos t)^{3} (s \sin t)^{3} \sin t $$ $$ \frac{\partial z}{\partial s} = 3s^{6} \sin^{4}t \cos^{3}t + 4s^{6}\sin{4}t\cos^{3}t = 7s^{6}\sin^{4}t\cos^{3}t $$

Similarly applying chain rule for $ \frac{\partial z}{\partial t} $ $$ \frac{\partial z}{\partial t} = \frac{\partial z}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial z}{\partial y} \frac{\partial y}{\partial t} $$ $$ = \Big( \frac{\partial}{\partial x} \ x^{3}y^{4} \Big) \Big( \frac{\partial}{\partial s} s \cos t \Big) + \Big( \frac{\partial}{\partial y} x^{3}y^{4} \Big) \Big( \frac{\partial}{\partial t} s \sin t \Big)$$ $$ \frac{\partial z}{\partial t} = (3x^{2}y^{4})(-s \sin t) + (4x^{3}y^{3})(s \cos t) = -3x^{2}y^{4}s \sin t + 4x^{3}y^{3} s \cos t $$

Now plugin $ x = s \ cos t \ text{and} \ y = s \sin t $ $$ \frac{\partial z}{\partial t} = -3(s \cos t)^{2}(s \sin t)^{4} s \sin t + 4(s \cos t)^{3}(s \sin t)^{3} s \cos t $$ $$ \frac{\partial z}{\partial t} = -3s^{7} sin^{5}t \cos^{2}t + 4s^{7}\sin^{3}t \cos^{4} t $$
