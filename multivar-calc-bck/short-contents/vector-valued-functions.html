Vector-Valued Functions

A real valued function y = f(x) assigns to each x from the domain $D\subset R$ a real value y from the codomain $B\subset R$. The equation of a line in 2D-space has the form $$ x = x_0 + t_a \ \ \text{,} \ \ y = y_0 + t_b $$ which can also be written in the vector form $$ \vec{r} = \vec{r_{0}} + t\vec{v} $$

We can regard $ \vec{r} = \vec{r_{0}} + t\vec{v} $ as a function which assigns to each t a vector $\vec{r} = \vec{r_{0}} + t\vec{v} $. This type of function is called a vector valued function. A vector valued function in 2D space is as follows $$ \vec{r}t = \begin{bmatrix} x(t) \\ y(t) \end{bmatrix} = x(t)\vec{i} + y(t)\vec{j} $$ A vector valued function in 3D space $$ \vec{r}t = \begin{bmatrix} x(t) \\ y(t) \\ z(t) \end{bmatrix} = x(t)\vec{i} + y(t)\vec{j} + z(t)\vec{j} $$

Example : Find the domain of the vector function $$ r(t) = \begin{bmatrix} \ln(t - 1) \\ t^2 \\ \frac{1}{9 - t^2} \end{bmatrix} $$ We need to find the domain of each components in the vector valued function. The domain of $\ln(t - 1)$ is $ \\ t - 1 > 0 \ \ t > 1 $ The domain of $t_2$ is all real numbers The domain of $ \frac{1}{9 - t^2} $ is $ \\ 9 - t^2 \neq 0 \\ - t^2 \neq -9 \\ t^2 \neq 9 \\ t \neq \pm 3 $

The first component restrics the domain of an entire vector function to $t > 1$. The sencond component does not restrict the domain at all. The third component tells us that the domain can not be +3 or -3. Hence we can say that the domain of the vector function is $ t > 1 \ \ \text{but} \ \ t \neq 3 $

Example : Identify the curve given by the following parameterization: $$ \vec{r(t)} = \begin{bmatrix} \cos(2t) \\ \sin(2t) \end{bmatrix} \ text{,} \ t \in [0, 2\pi]$$ $$ let x(t) = \cos(t) \ \text{and} \ let y(t) = \sin(t) $$ $$ \cos(2t) = \cos^{2}t - \sin{2}^t $$ $$ \cos(2t) = \cos^{2}t + \sin{2}^t - \sin{2}^t - \sin{2}^t $$ $$ \cos(2t) = 1 - 2\sin{2}^t $$ $$ x = 1 - 2y^2 $$ when y = 0, x is 1. Now we can plugin x = -1 in this equation $$ -1 = 1 - 2y^{2} => 2y^{2} - 2 = 0 $$ $$ 2(y - 1)(y + 1) = 0 $$ So we can see indeed y = 1 or y = -1. Since the equation we get is $x = 1 - 2y^2$ it is indeed a parabola.

Limits, Continuity and Derivatives

The concept of limit means approaching. Limit represents that how the function f(x) behaves as it gets closer and and closer(approaches) to some value. $$ \lim_{t \to C} f(x) $$

The notions of limit, continuity and derivative for real-valued functions are all applicable to vector-valued functions through the constituent components.

Finding the Limit of a Vector valued function

$$ \large{\lim_{t \to 2}[(t^2 - 2)i + \ln(t + e)j + \frac{4t}{sint}k]} $$

Taking the limit of each term seperately $$ \large{\lim_{t \to 2}(t^2 - 2)i + \lim_{t \to 2}\ln(t + e)j + \lim_{t \to 2}\frac{4t}{sint}k} $$ Evaluating first two terms as t -> 0 we get $\lim_{t \to 2}(0^2 - 2)i + \lim_{t \to 2}\ln(0 + e)j + \lim_{t \to 2}\frac{4t}{sint}k \\ \lim_{t \to 2}(0^2 - 2)i + \lim_{t \to 2}\ln(0 + e)j + \lim_{t \to 2}\frac{4t}{sint}k \\ (0^2 - 2)i + ln(0 + e)j + \lim_{t \to 2}\frac{4t}{sint}k \\ -2i + ln(e)j + \lim_{t \to 2}\frac{4t}{sint}k \\ -2i + 1j + \lim_{t \to 2}\frac{4t}{sint}k \\ -2i + j + \lim_{t \to 2}\frac{4t}{sint}k $

As the last term is giving 0/0 when t -> 0 we will use L'Hospital's rule replacing the numerator and denominator with their derivatives $ -2i + j + \frac{4}{cos0}k \\ -2i + j + 4k $

$ -2i + j + 4k $ is the limit of the vector function.
