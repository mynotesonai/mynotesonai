Arc Length

In single variable Calculus if $x'(t)$ and $y'(t)$ are continuous, then the curve given by $$ x = x(t), y = y(t), a \geq t \geq b \ \text{has arc length} \ \ L = \displaystyle \int^{b}_{a} \sqrt{x'(t)^{2} + y'(t)^{2}} \ dt $$ This formula generalizes to 3-space curves. The Arc length of the curve $$ x = x(t), y = y(t), z = z(t), a \geq t \geq b $$

Example : Find the Arc length of the curve $\vec{r} = a \cos t \vec{i} + a \sin t \vec{j} \ \ , 0 \leq t \leq 2\pi $

Solution : $$ L = \displaystyle \int^{2\pi}_{0} \sqrt{x'(t)^{2} + y'(t)^{2}} \ dt $$ $$ = \displaystyle \int^{2\pi}_{0} \sqrt{(-a \sin t)^2 + (-a \cos t)^2} \ dt $$ $$ \displaystyle \int^{2\pi}_{0} \sqrt{a^2(\sin^2(t) + \cos^2(t)) \ dt} $$ $$ \displaystyle \int^{2\pi}_{0} \sqrt{a^2} /dt $$ $$ = 2 \pi a $$

Arc Length as a Parameter

As a curve as a path of a moving object the same trajectory can be travelled at a different speed. Hence the same trajectory that is depicted by the curve can be represented in the parametric form with different parameters. The curve has different parametric equations. $$ x(t) = a \cos t \ \ \ y(t) = a \sin t \ \ \ 0 \leq t \leq 2\pi $$ $$ x(s) = a \cos(s^2) \ \ \ y(s) = a \sin(s^2) \ \ \ 0 \leq s \leq 2\sqrt{2\pi} $$ $$ x(u) = a \cos(2\pi e^{-u}) \ \ \ y(u) = a \sin(2\pi e^{-u}) \ \ \ 0 \leq u \leq \infty $$
