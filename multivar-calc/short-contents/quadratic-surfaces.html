Quadratic Surfaces

Quadratic Surfaces are three dimensional extension of a conic section.

Quadratic surfaces are the graphs of equations that can be expressed in the form: $$ Ax^{2} + By^{2} + Cz{2} + Dxy + Exz + Fyz + Gx + Hy + Jz + k = 0 $$ When Quadratic surfaces intersects the coordinate plane(xy-plane, xz-plane, or yz-plane) we get a conic section such as Line, Parabola, Circle, Ellipse, and Hyperbola.

Equations for different Quadratic Surfaces

Circle
$$ \large{x^{2} + y^{2} = r^{2} \ \ \text{or} \ \ \frac{x^{2}}{r^{2}} + \frac{x^{2}}{r^{2}} = 1} $$
Cylinder
$$ \large{\frac{x^{2}}{r^{2}} + \frac{x^{2}}{r^{2}} = 1} $$
Ellipse
$$ \large{\frac{x^{2}}{a^{2}} + \frac{y^{2}}{b^{2}} = 1} $$
Ellipsoid or Sphere
$$ \large{\frac{x^{2}}{a^{2}} + \frac{y^{2}}{b^{2}} + \frac{z^{2}}{c^{2}}= 1} $$
Ellipse
$$ \large{\frac{x^{2}}{a^{2}} + \frac{y^{2}}{b^{2}} = 1} $$
Cone
$$ \large{\frac{x^{2}}{a^{2}} + \frac{y^{2}}{b^{2}} = \frac{z^{2}}{c^{2}} \ \ \text{or} \ \ \frac{x^{2}}{a^{2}} + \frac{y^{2}}{b^{2}} - \frac{z^{2}}{c^{2}} = 0} $$
Parabola
$$ \large{ax^{2} + by = c \ \ \text{or} \ \ ax + by^{2} = c} $$
Parabolloid
$$ \large{\frac{z}{c} = \frac{x^{2}}{a^{2}} + \frac{y^{2}}{b^{2}} \ \ \text{or} \ \ \frac{x}{a} = \frac{y^{2}}{b^{2}} + \frac{z^{2}}{c^{2}} \ \ \text{or} \ \ \frac{y}{b} = \frac{x^{2}}{a^{2}} + \frac{z^{2}}{c^{2}} } $$
Hyperbola
$$ \large{\frac{x^{2}}{z^{2}} - \frac{y^{2}}{b^{2}} = 1 \ \ \text{or} \ \ \frac{y^{2}}{b^{2}} - \frac{x^{2}}{a^{2}} = 1 } $$

Hyperboloid of one Sheet
$$ \large{\frac{x^{2}}{a^{2}} + \frac{y^{2}}{b^{2}} - \frac{z^{2}}{c^{2}} \ \ \text{or} \ \ \frac{x^{2}}{a^{2}} - \frac{y^{2}}{b^{2}} + \frac{z^{2}}{c^{2}} \ \ \text{or} \ \ \frac{y^{2}}{b^{2}} + \frac{y^{2}}{b^{2}} + \frac{x^{2}}{s^{2}} = 1 }$$
