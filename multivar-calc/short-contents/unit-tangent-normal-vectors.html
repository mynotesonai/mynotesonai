Unit Tangent and Normal Vectors

Unit tangent, unit normal and Binormal vectors play pivotal role in understanding the motion of an object.

Unit Tangent Vector

If we let $ C $ be a smooth curve with position vector $ \vec{r(t)} $, then the Unit Tangent Vector $ \vec{T{t}}$, is defined to be $ \vec{T{t}} = \frac{\vec{r{t}}}{\vec{r'{t}}} $$ and represents the unit vector in the direction of the velocity vector.

Unit Normal Vector

Unit normal vactor is denoted by $ \vec{N(t)} is defined as $ \vec{N(t)} = \frac{\vec{T'{t}}}{||\vec{T'{t}}||}, and is orthogonal to the unit tangent vector and the curve and the points in the direction where the curve is bending.

BiNormal Vector

If we let $ C $ be a smooth curve with position vector $ \vec{r(t)} $, then the binormal vector denoted by $ \vec{B(t)} $ is defined to be $ \vec{B(t)} = \vec{T(t)} \times \vec{N(t)}$. Binormal vector is a unit vector that is perpendicular to both $ \vec{T(t)} \ \text{and} \ \vec{N(t)} $
