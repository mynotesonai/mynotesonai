Stokes Theorem

Stokes' theorem is similar to Green's theorem for higher dimensions. Green's theorem is used to convert a line integral to a double integral. Stokes' theorem is used to convert a line integral to a surface integral.

Stokes' theorem relates the line integral to surface integral. $$ \displaystyle \int_{c}^{} F.dr - \displaystyle \int \displaystyle \int_{s} curl F.dS $$

Example Find $ \displaystyle \int \displaystyle \int_{s}^{} curl F. dS. S \ \text{is the hemisphere} \ x^{2} + y^{2} + z^{2} = 9, $

$ F(x, y, z) = 2y \cos z \ i + c^{x} \sin z \ j + xe^{y} \ k \text{,} z \geq 0 $, upward orientation

$ \text{Stokes' Theorem} - \displaystyle \int \displaystyle \int_{s}^{} curl F. dS = \displaystyle \int_{c} F. dr $

$ C $ in the above graph runs all along the bottom of the hemisphere. where z equals 0

if we want to get the curve $ C $ we can plugin $z=0 \ \ C$ is given by $x^{2} + y^{2} = 9$ z is equal to 0

r is the vector function in the formula, we would need to parameterize this equation so we can get it in terms of vector function. The equation that we have here is a circle with a radius of 3. That means we can parameterize this as a vector function as follows

$ \vec{r(t)} = \begin{bmatrix} 3 \cos t, & 3 \sin t, 0 \end{bmatrix} $

$ \vec{r'(t)} = \begin{bmatrix} -3 \sin t, & 3 \cos t, 0 \end{bmatrix} $

The next step is to plug-in the vector function $\vec{r(t)}$ and plug this function into $ F(x, y, z) $

$ F(\vec{r(t)}) = \begin{bmatrix} 2(3\sin t) \cos 0, e^{3 \cos t} \sin 0, (3 \cos t) e^{3 \sin t} \end{bmatrix} $

$ = \begin{bmatrix} 6 \sin t, 0, (3 \cos t)e^{3 \sin t} \end{bmatrix} $

Since this is a circle the interval here is from $ 0 $ to $ 2\pi $

$ = \displaystyle \int_{0}^{2\pi} F(r(t)) . r'(t) \ dt $

Here we have to take the dot product between $ F(r(t)) $ and $ r'(t) $

$ = \displaystyle \int_{0}^{2\pi} \begin{bmatrix} 3 \cos t, & 3 \sin t, 0 \end{bmatrix} . \begin{bmatrix} -3 \sin t, & 3 \cos t, 0 \end{bmatrix} $

$ = \displaystyle \int_{0}^{2\pi} -18 \sin^{2} t \ dt $

$ = \displaystyle \int_{0}^{2\pi} -18 [\frac{1}{2}(1 - \cos (2t))] \ dt $

$ = -9 \displaystyle \int_{0}^{2\pi} 1 - \cos (2t) \ dt $

$ = -9 \left[ t - \frac{1}{2}\sin (2t) \right]_{0}^{2\pi} \ dt $

$ = -9 [2 \pi] $

$ = -18 \pi $

Hence $ \displaystyle \int \displaystyle \int_{s}^{} $ curl $ \ F. dS. \ S $ is the hemisphere $ x^{2} + y^{2} + z^{2} = 9 $ is $-18 \pi $
